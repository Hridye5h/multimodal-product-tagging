Multimodal Product Tagging

One CLIP model reads a product's image + title and auto-tags it with four catalog attributes at once — category, colour, gender, season — and stays accurate even when the title is missing.

CLIP (ViT-B/32) PyTorch Multi-task DistilBERT + ResNet-50 Robustness analysis 30K products

What it does

E-commerce catalogs need every product tagged with many attributes — that tagging is normally manual. This model does it automatically from the product photo and its title in a single forward pass, using CLIP's aligned vision-language embeddings with one classification head per attribute.

The interesting part — robustness to missing metadata

Real catalogs have sparse or junk titles. A model that leans only on text breaks when the title is gone. By training with random "missing title" augmentation, this model falls back on the image — so mean accuracy only drops from 92% → 82% when titles are removed, instead of collapsing.

The single-modality ablation (separate DistilBERT + ResNet-50) makes the point sharply:

ModelWith titleTitle removed
Text-only (DistilBERT)0.9750.028
Image-only (ResNet-50)0.9050.905
Fusion (both)0.9800.886

When titles vanish, text-only collapses to ~3% while fusion holds ~89% — a +86-point robustness gain by using the image.

Reproducible Kaggle notebooks, modular PyTorch source (data / model / training split into importable modules), discriminative-LR fine-tuning, a synthetic-data fallback so the pipeline runs without the dataset, and a comma-safe CSV reader for the messy real metadata.
